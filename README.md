# Оценка качества застройки жилых кварталов
## Курсовая работа по дисциплине «Базы данных» / «Моделирование данных»

# Описание проекта
## Данный проект представляет собой комплексную систему анализа качества жилых кварталов, основанную на оценке компаний-застройщиков по пяти ключевым критериям:

Цена (доступность жилья)
Благоустройство (инфраструктура двора и района)
Экологичность (озеленённость, чистота воздуха)
Близость к центру (транспортная доступность)
Качество застройки (архитектура, материалы, надёжность)
Все данные хранятся в реляционной базе данных PostgreSQL, обрабатываются с помощью SQL-запросов и анализируются через скрипты на JavaScript (Node.js).
